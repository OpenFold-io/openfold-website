What is OpenFold?

Mission Statement

The incredible performance of AlphaFold 2 is currently restricted to academic users and completely inaccessible for use toward commercial applications. However, as academic groups begin to develop derivative tools, the need to bring an industry-compatible version of this tool into fruition is clear. With the recent publication of a detailed methods paper as well as MIT-licensed source code for interacting with the models, which were trained on publicly available data, there is a path to reproduce DeepMindâ€™s work in an industry-friendly form

Goals

The aim of this work is to

  • Develop a permissively licensed trained model that performs similar to AlphaFold 2
  • Provide the tools used to train the model under the same license
  • Host the model files for one year
